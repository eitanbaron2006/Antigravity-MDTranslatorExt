# Test Markdown File

This is a test markdown file to demonstrate the translation functionality.

## Features

- Automatic translation on file open
- Side-by-side view of original and translated content
- Summary generation
- Configuration options for language and service

## Usage

Open this file in VS Code or Google Antigravity with the Markdown Translator extension installed.

## Translation Services

The extension supports:

1. Google Translate
2. OpenAI
3. Google Gemini

## Installation

1. Build the extension
2. Install from VSIX
3. Configure settings
4. Open any markdown file

## Summary

This test file shows how the extension automatically translates and summarizes markdown content without modifying the original file.
